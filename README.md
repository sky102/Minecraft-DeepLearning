# Deep Learning With Minecraft 
Tabular Q-learning is uitilized to train a Minecraft agent to navigate the environment to reach a destination.

# Usage
To run the code:
1) Install malmo

2) Run minecraft client

3) Run the project code in src directory

On Windows: 
python runAgent.py

On Macs:
python3 runAgent.py
